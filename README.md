### ✨ A clean and responsive web layout built using **Bootstrap 4**, focusing on modern container-based design.

## 🚀 This project demonstrates the effective use of:

* `.container` for fixed-width responsive layouts
* `.container-fluid` for full-width designs
* Bootstrap grid system (rows & columns)

## 🎯 **Key Features**

* 📱 Fully responsive design
* 🎨 Clean and minimal UI
* ⚡ Beginner-friendly structure
* 🧩 Easy to understand layout system

# 💡 This project is perfect for students and beginners who want to learn how to build structured and responsive web pages using Bootstrap.

## 🔧 **Technologies Used**

* HTML5
* Bootstrap 4
